# Smart Expense Tracker - Frontend

A premium, production-ready MERN stack frontend built with React, Vite, React Router, and Context API.

## Tech Stack
- **Framework:** React & Vite
- **Routing:** React Router v6
- **State Management:** Context API (AuthContext, ThemeContext)
- **Styling:** Premium Glassmorphic Vanilla CSS (responsive, mobile-first, dark mode optimized)
- **Icons:** Lucide React
- **Data Visualizations:** Recharts

## Getting Started

### Installation
1. Navigate to this directory:
   ```bash
   cd "C:\Users\SUMIT\OneDrive\Dokumen\frontend part of expense tracker\frontend"
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Key Features & UI Aesthetics
- **Dual Mode Connectivity:** Auto-detects if the backend is offline/unreachable and falls back to LocalStorage mock states, letting the user test the full functionality.
- **Glassmorphic Theme:** Dark mode default using custom blur backdrops, gradient glow accents, and responsive layout grids.
- **OCR Receipt Scanning:** Drag-and-drop receipt image uploader with full details parser.
- **Multi-wallet / Budget Alerts:** Custom alerts triggering when categories exceed 80% and 100% threshold limits.
